# Contribution Guide

## Commit Message Style

```html
<type>(<scope>): <subject>
<BLANK LINE>
<body>
<BLANK LINE>
<footer>
```

### Allowed types

* feature (feature)
* fix     (bug fix)
* docs    (documentation)
* style   (formating, missing semi colons, ...)
* refactor
* test    (when adding missing tests)
* chore   (maintain)

### Available scopes

* command
* db
* server
* resp
* context
* conf
* metric
